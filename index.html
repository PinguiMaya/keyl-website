How to use Keyl

01

Create your
own KEY

With your own key, only you will have access to encrypt and decrypt that file, the key is unique so don't lose it.

02

Select your
own KEY

After you have your own key, select it in the program. So KeyL can process your files

03

Select your
.TXT file

KeyL works with .txt files, but you can convert your word, pdf files to txt so you can encrypt it. KeyL selected the .TXT file type since it is a very easy file type to encrypt with great security, but KeyL will also have versions for .docx .pdf files, although work is still ongoing on that version.

04

Select the number of times
the file will be processed

Apart from having your key, you have to remember a number from 1 to 30 to be able to decrypt your file, this number is the type of encryption security (Low: 1-10, Normal: 10-20 High: 20-30)

05

Encrypt your
FILE

KeyL will encrypt your file very securely, using your unique key and the number of times you want it to be processed as explained above.

06

Decrypt your
FILE

KeyL will decrypt your file very securely, using your unique key and the number of times you want it to be processed as explained above.
